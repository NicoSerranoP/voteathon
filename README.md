# Start the app locally

### 1 Build the files

```shell
yarn build
```

### 2 Start a node

```shell
yarn contracts hardhat node
```

### 3 Deploy smart contracts

in new terminal window, from root:

```shell
yarn contracts deploy
```

### 4 Start a relayer (backend)

```shell
yarn relay start
```

### 5 Start a frontend

in new terminal window, from root:

```shell
yarn frontend start
```
